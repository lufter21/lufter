Веб-разработка. Создание темы WordPress

Качественная веб-разработка вашего проекта. Я не делаю дизайн сайта, работаю исключительно с предоставленными макетами.

Я выполняю:

  • HTML-верстку сайта, frontend разработку: делаю из картинки работающий сайт
  • Программирование на стороне сервера: например, отправка данных из формы на e-mail
  • Верстку и создание темы для WordPress: создаю тему для WordPress с вашим дизаном
  • Cоздание темы WordPress из ваших HTML-шаблонов

Я не работаю по принципу — "Сначала сделайте, а потом мы оплатим". Сотрудничаю исключительно через фриланс биржу с заключением безопасной сделки: средства замораживаются на счету биржи, после успешного выполнения проекта — переводятся мне, либо возвращаются вам, если проект не выполнен. Это гарантирует вам, что проект будет выполнен. Это гарантирует мне, что работа будет оплачена. Все коммисси биржи оплачиваю я.
Предлагаю такие биржи: upwork.com, fl.ru или weblancer.net, также вы можете предложить удобную вам биржу, с функцией безопасной сделки.

HTML-верстка, frontend разработка

Вы предоставляете дизайн — я делаю из "картинки" работающий сайт. Макеты должны быть в формате .psd или выполнены в zeplin, figma.

Естественно я не ограничиваюсь просто версткой, кроме этого выполняю анимации, различные эффекты и программирование на стороне клиента. Вы получите шаблон с работающими интерактивными элементами: формы с мгновенной проверкой ввода, всплывающие окна, слайдеры, калькуляторы, таймеры, диаграммы, графики, встраивание функционала сторонних сервисов и многое другое. Также выполняю адаптацию, адаптивную верстку: шаблон будет адаптироваться под разные размеры экрана, от смартфона до большого монитора.

Вы получите сайт, который уже можно будет наполнить контентом, загрузить на хостинг и открыть в браузере.

Программирование, backend

Выполняю программирование на стороне сервера, на языке php. Например, отправку данных из формы на e-mail, чтение из базы данных, запись в базу данных, парсинг, интеграцию сторонних сервисов и многое другое.

Создание темы для WordPress

Если сайт небольшой, например визитка на 5-10 страниц — можно обойтись без CMS (систем управления контентом). Но если это блог или новостной сайт, где страниц может быть очень много — без CMS управлять сайтом будет очень сложно. Одной из самых популярных CMS является WordPress.

Я создаю тему для WordPress по вашему макету, предварительно сделав HTML-шаблон, или из ваших готовых HTML-шаблонов.
